8. JavaScript 문법 - Template literal
문자열이 너무 길때 코드 내에서 줄바꿈을 하고싶다면 역슬래시'\'를 붙이고 줄을 바꿔주자
이때 프로그램을 실행하면 역슬래시는 없는 문자처럼 취급하기 때문에 띄어쓰기 또한 없다.
\n으로 해준다면 실제 프로그램 실행시에도 줄바꿈이 표현된다.
그런데 이런것이 실제로는 불편하기 때문에 node.js에는 Template literal이 있다.
literal = 정보를 표현하는 기호, Template literal을 표현하고 싶으면 ''대신에
``를 사용해준다. 이렇게 되면 역슬래시를 이용하지 않고도 코드 내에서 줄바꿈이 가능하다.
변수를 집어넣는 경우도 다움표를 닫고 +를 사용하지 않고 ${변수}를 해준다. 변수 대신에도
만약 ${1+1}을 해주어도 실행하면 2가 나온다.
